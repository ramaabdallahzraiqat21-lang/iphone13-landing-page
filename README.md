This project is a modern and interactive landing page for the iPhone 13 Pro Max, built using HTML, CSS, and JavaScript.

Features
- Clean and minimal design inspired by Apple
- Interactive color selection for the product
- Dynamic image and background color changes using JavaScript
- Smooth hover effects and transitions
- Responsive layout for different screen sizes

 Technologies Used
- HTML
- CSS
- JavaScript

 Project Overview
This website allows users to explore the iPhone 13 Pro Max by selecting different colors. When a user clicks on a color option, the product image and background update dynamically using JavaScript, providing a more interactive user experience.

 Purpose
The goal of this project is to demonstrate front-end development skills, including DOM manipulation, responsive design, and creating interactive UI without using any frameworks or backend technologies.
